The personal portfolio and digital playground of **Sanjay M** — AI & Data Science Engineer.

Designed with a focus on editorial typography, fluid motion, and premium micro-interactions to reflect a high standard of software craftsmanship.

## 🚀 Tech Stack

*   **Framework:** React 19 + Vite
*   **Motion & Animations:** Framer Motion
*   **Smooth Scrolling:** Lenis
*   **Styling:** Custom CSS Design System (CSS Variables, BEM Methodology)
*   **Typography:** Playfair Display, DM Sans, JetBrains Mono

## 🧠 Design Philosophy

This portfolio breaks away from standard "AI-generated" or templated layouts. It relies on:
1.  **Intentional Motion:** Every animation (scroll reveals, hover states, counters) has a purpose—to guide attention and provide satisfying feedback.
2.  **Editorial Aesthetics:** A warm, refined color palette and stark typography that feels more like a high-end magazine than a standard SaaS dashboard.
3.  **Performance:** Built on Vite for lightning-fast HMR and optimized production builds.

## 🛠️ Running Locally

To run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SanjayMuthuswamy/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`
